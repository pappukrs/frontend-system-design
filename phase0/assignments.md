# Phase 0 Assignments: Foundations

## Assignment 1: NFR Definition
**Scenario:** You are the Lead Architect for a new "Twitter-like" microblogging platform.
**Task:** List the top 5 Non-Functional Requirements (NFRs) for this app. For each, explain:
1.  Why it is critical.
2.  One specific metric or tool you would use to measure it.

## Assignment 2: The "Offline-First" Mental Model
**Task:** Draw (or describe) the user flow if a user tries to "Like" a post while their internet connection just dropped.
- How do you handle the UI?
- How do you handle the data sync once the connection returns?
- What happens if the user closes the tab before the sync completes?

## Assignment 3: Trade-off Analysis

- SEO
- First Input Delay (FID)
- Server Cost
- Interactive-ness (Time to Interactive)
